FAQ

How do I connect to App Store Connect?

Go to Settings in the sidebar, enter your API Key ID, Issuer ID, and paste your .p8 private key. You can create an API key at App Store Connect → Users → Integrations → API.

Is my API key safe?

Yes. Your credentials are stored in the macOS Keychain on your device. They are never sent to any server other than Apple's API. The app has no analytics, no telemetry, and no third-party SDKs.

How do I capture simulator screenshots?

Go to Screenshots in the sidebar. The Simulator Capture panel automatically detects running iOS Simulator windows. Click "Capture Screenshot" to grab a screenshot at Retina resolution, then "Send to Composer" to design App Store graphics.

How does the AI Agent API work?

Toggle the API server on from the Terminal panel (bottom bar). External AI agents (Claude Code, Cursor, etc.) can then call http://localhost:29117/api/apps and other endpoints. All activity is visible in the Agent Activity tab in real time.

Can I use this with multiple App Store Connect accounts?

Yes. The app supports multiple ASC accounts. Switch between them in Settings.

What macOS version is required?

macOS 14 (Sonoma) or later. Simulator capture requires macOS 12.3+ (ScreenCaptureKit).

How do I get a refund?

If you purchased from the Mac App Store, request a refund through reportaproblem.apple.com. If you purchased directly from this website, email us within 30 days for a full refund.
