Frequently Asked Questions

Where do I get my App Store Connect API credentials?

Go to App Store Connect → Users and Access → Integrations → App Store Connect API. Create a new key with Admin access. You'll need the Key ID, Issuer ID, and the downloaded .p8 private key file.

Is my API key stored securely?

Yes. Your credentials are stored in the macOS Keychain, encrypted at rest. They are never sent anywhere except Apple's App Store Connect API.

Can I use this app with multiple App Store Connect accounts?

Currently the app supports one set of credentials at a time. You can switch credentials via the Settings section.

What is the safeguard password?

The safeguard password is a local password you set during onboarding. It protects dangerous actions like "Submit for Review" from being triggered accidentally. It's stored as a hash locally and never transmitted.

What is the MCP server?

The MCP (Model Context Protocol) server is an optional companion CLI that lets AI coding assistants like Claude Code manage your App Store Connect workflows programmatically. It's distributed separately from the Mac App Store app.

The app shows "Unable to Load Apps"

Check that your API credentials are correct in Settings. Ensure your API key has Admin access and hasn't been revoked in App Store Connect.
