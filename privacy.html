Privacy Policy

Last updated: April 1, 2026

Summary

Simple App Shipper does not collect, store, or transmit any personal data. Your App Store Connect credentials are stored locally on your Mac using the macOS Keychain. No analytics, no tracking, no telemetry.

Data Collection

We collect zero data. The app:

  • Does not send analytics or crash reports to any server
  • Does not use cookies or tracking pixels
  • Does not include any third-party SDKs that collect data
  • Does not phone home or check licenses via network

App Store Connect Credentials

Your API Key ID, Issuer ID, and Private Key are stored exclusively in the macOS Keychain on your device. They are never transmitted to any server other than Apple's App Store Connect API (api.appstoreconnect.apple.com) for the purpose of performing actions you explicitly request.

Local API Server

The app includes an optional localhost-only HTTP server (bound to 127.0.0.1) for AI agent integration. This server is only accessible from your own machine and is never exposed to the network. It does not transmit data externally.

Screen Capture

The simulator capture feature uses Apple's ScreenCaptureKit framework. Captured screenshots and recordings are stored in your local temporary directory and are never uploaded anywhere unless you explicitly choose to upload them to App Store Connect.

Third-Party Services

The app communicates only with:

  • Apple App Store Connect API (api.appstoreconnect.apple.com) — to manage your apps, using credentials you provide
  • Apple iTunes Search API (itunes.apple.com/search) — public API for ASO keyword research, no authentication required

No other third-party services are contacted.

Children's Privacy

This app is a developer tool and is not directed at children under 13.

Changes

If this policy changes, the updated version will be posted here with a new date.
